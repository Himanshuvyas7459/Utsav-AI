🎉 Utsav AI – Smart AI Event Management Platform

Utsav AI is a full-stack MERN-based event management platform designed to simplify event planning and enhance user experience using intelligent features. It allows users to discover, manage, and organize events efficiently with a modern UI and powerful backend.

🚀 Features
🔐 User Authentication (JWT based login/signup)
📅 Event Creation & Management
🤖 AI-Powered Event Suggestions
🧑‍🤝‍🧑 Organizer Request System
🛒 Ticket Booking System (if implemented)
📊 Admin Dashboard
📱 Fully Responsive UI
⚡ Fast API with Express & MongoDB
🛠️ Tech Stack

Frontend:

React.js
Tailwind CSS / CSS
Axios

Backend:

Node.js
Express.js

Database:

MongoDB

Other Tools:

JWT Authentication
REST APIs
Cloud Deployment (Vercel / Render)
